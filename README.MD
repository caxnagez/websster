# Emergency Call Tracking System

## Описание

Веб-приложение на Flask для отслеживания экстренных вызовов. Диспетчер может создавать вызовы и назначать их нескольким департаментам (медицинская, пожарная, полиция). Каждый департамент может обновлять статус своего участия в вызове. Реализован RESTful API для взаимодействия с системой.

## Функциональность

- Регистрация и аутентификация пользователей.
- Ролевая модель:
  - **Admin**: Просмотр всех вызовов, создание/редактирование/удаление вызовов.
  - **Dispatcher**: Просмотр всех вызовов, создание новых вызовов.
  - **Medical/Fire/Police**: Просмотр назначенных вызовов, обновление статуса своего участия.
- Создание, просмотр, редактирование и удаление вызовов.
- Отслеживание статуса вызова по департаментам.
- RESTful API для всех основных операций.
- Тестирование API с помощью `unittest`.
- **JavaScript**: Интерактивное обновление статуса департамента через AJAX (без перезагрузки страницы).
- Использование Bootstrap 5 для стилизации.

## Установка

1.  Клонируйте репозиторий:

    git clone https://github.com/caxnagez/websster

2.  (Опционально) Создайте и активируйте виртуальное окружение:

    python -m venv venv
    
    source venv/bin/activate

3.  Установите зависимости:

    pip install -r requirements.txt

## Запуск приложения

python app.py

## Запуск тестов

**Перед запуском убедитель, что Flask-сервер не работает (app.py не запущен), тесты работают через изоляцию бд**

python tests.py
